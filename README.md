# Delivery Orders Analysis

Análisis de órdenes tipo delivery usando Python, pandas y CSV.

## Qué hace

- Lee un archivo CSV con órdenes.
- Analiza datos con pandas.
- Cuenta órdenes por ciudad.
- Calcula facturación por ciudad.
- Cuenta órdenes por estado.
- Genera reportes automáticos en CSV.

## Reportes generados

- `output/orders_by_city.csv`
- `output/revenue_by_city.csv`
- `output/orders_by_status.csv`
