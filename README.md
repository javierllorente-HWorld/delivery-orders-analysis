# Delivery Orders Analysis

Proyecto simple en Python para analizar datos de órdenes de delivery desde archivos CSV.

Es la segunda etapa de un roadmap técnico personal de 8 proyectos que armé para repasar bases, practicar con ejemplos concretos y mejorar mi lado técnico paso a paso.

La idea de este proyecto fue avanzar un poco más sobre el trabajo con datos: leer órdenes, agrupar información, calcular métricas simples y generar reportes.

---

## Qué hace

* Lee un archivo CSV con órdenes.
* Analiza datos usando pandas.
* Cuenta órdenes por ciudad.
* Calcula facturación por ciudad.
* Cuenta órdenes por estado.
* Genera reportes CSV de salida.

---

## En qué me enfoqué

* Agrupar información y calcular métricas simples.
* Generar reportes automáticos.
* Ordenar archivos de entrada y salida.
